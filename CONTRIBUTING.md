# Contributing to MicronOS

Thank you for your interest in contributing to MicronOS!

## How to Contribute

1. **Fork the repository** on GitHub
2. **Clone your fork** locally
3. **Create a feature branch**: `git checkout -b feature/my-feature`
4. **Make your changes** following our coding standards
5. **Add tests** for your changes
6. **Run the test suite**: `cargo test --workspace`
7. **Commit your changes**: `git commit -m "Add feature: description"`
8. **Push to your fork**: `git push origin feature/my-feature`
9. **Open a Pull Request** on GitHub

## Coding Standards

- Follow Rust idioms and best practices
- Use `cargo fmt` for code formatting
- Run `cargo clippy` before submitting
- Write documentation in English
- Add tests for all public APIs

## Code of Conduct

Please be respectful and constructive in all interactions.
